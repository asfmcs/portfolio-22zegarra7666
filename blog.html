Entry #1
"Hello World!" Is the first thing we learnt how to write in HTML. In Computers Class we've been learning many, many things in just a few weeks. I've come a far way, going from no knowledge what so ever, to easily being able to format a website (altough simple, very complex) with different colors, pictures, fonts, and links within. I've come a long way from not even knowing what HTML or CSS even meant or was. I've learnt about 'classes' and 'ID's' (knowing how to modify a specific), I've also learnt about formatting my html itself, so that it is readble, understandable and legible not only by the computer but the teacher and myself as well.

In this unit we first started with some lessons in Khan Academy that taught us the most basic things, like how to write a heading, the different sizes of headings, going from the biggest size (h1) to the smallest size (h6). Then we learned how to write text, which is just a simple "p" in between arrow brackets "<>". Then we learned many other things like formatting a page, making space (aka breaks) in between our paragraphs or any content what so ever. We also learnt how to add color o our backgrounds, our text, changing the color of the text itself, adding pictures and then applying our knowledge into a simple website fully created and programmed by us.
